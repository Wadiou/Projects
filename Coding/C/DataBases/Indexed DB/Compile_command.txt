To Run IndexedDB:

    Navigate to the output folder in the terminal:
    cd output
    Run the executable (Mac/Linux):
    ./IndexedDB
    Or directly click on it on Windows


To Compile and Run IndexedDB:

    Navigate to the Indexed DB folder in the terminal:
    cd "Indexed DB"
    Compile and run using the following command:
    g++ Main.cpp Indexing.cpp Generator.cpp File_Functions.cpp Secondary_Functions.cpp List.cpp -o output/IndexedDB ; ./output/IndexedDB
